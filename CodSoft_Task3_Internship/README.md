# Iris Flower Classification using Machine Learning

## Overview

This project focuses on classifying Iris flowers into three different species: Iris-setosa, Iris-versicolor, and Iris-virginica using Machine Learning. The classification is performed based on the flower's sepal length, sepal width, petal length, and petal width measurements.

The Iris dataset is one of the most popular datasets in Machine Learning and is commonly used for introductory classification tasks.

## Objective

To build a Machine Learning model that can accurately predict the species of an Iris flower using its sepal and petal measurements.

## Dataset

The dataset contains 150 samples of Iris flowers with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

Species Classes:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

## Technologies Used

* Python
* Pandas
* Scikit-Learn

## Machine Learning Algorithm

* Random Forest Classifier

## Project Workflow

1. Load the Iris dataset.
2. Perform data preprocessing.
3. Split the dataset into training and testing sets.
4. Train the Random Forest Classifier.
5. Evaluate model performance.
6. Predict the species of new Iris flower samples.

## Results

* Successfully classified Iris flowers into their respective species.
* Achieved high prediction accuracy on the test dataset.
* Generated classification report and confusion matrix for evaluation.
