# CodSoft Internship Projects

This repository contains all projects completed during my CodSoft Data Science Internship.

## Projects

### Task 1 - Titanic Survival Prediction
Predict whether a passenger survived the Titanic disaster using Machine Learning.

### Task 2 - Movie Rating Prediction
Build a regression model to predict movie ratings based on historical movie data.

### Task 3 - Iris Flower Classification
Classify Iris flowers into different species using sepal and petal measurements.

### Task 4 - Sales Prediction
Predict future sales using advertising and marketing data.

### Task 5 - Credit Card Fraud Detection
Detect fraudulent credit card transactions using classification algorithms.
